基本html语法

代码结构

一个简单的html框架：

<!DOCTYPE html>
<html>

    <head>
        <title>
        title
        </title>

    </head>
    <body>
        <p>par1</p>
        <p>par2</p>
    </body>

</html>

标签

有序列表(Ordered List)：

<ol>
    <li>person 1</li>
    <li>person 2</li>
    <li>Matt Damon</li>
</ol>

无序列表标签是ul(Unordered List)。

注释代码：

<!-- This is an example of comment -->

标签属性

a标签中的href或img标签中的src都是属性，一般形式可以这么写：

<label attr="">content</label>

一些标签效果：

<p style = "color:red; font-size:20px; font-family:Garamond">some text</p>

输出:

some text

其它还有background-color,text-align

表格

table,tr,td,thead,tbody,th div(Short for division)用于进行网页的块设置。 span用于改变某部分文字的属性。

Cascading Style Sheets

为什么要将内容和表现分离？ 对多个元素赋予同样的表现形式 对多个HTML文件用一个CSS文件统一格式化

CSS的插入方法： 1. 用style标签（放在head标签内） 2. 独立文件，并在head标签内加入：

<link type="text/css" rel="stylesheet" href="stylesheet.css" />

CSS三要素：selector,property,value

p {
    color: red;
}

CSS注释：

/* This is an example comment */
